Template (para envio em massa)
Exatamente como está cadastrado no Meta Business Manager
Só preencha se o template foi aprovado com header de imagem. Caso contrário, deixe em branco.
Selecione conforme o template foi aprovado pela Meta. Templates novos costumam usar variáveis nomeadas.
Para cada variável {{nome_da_var}} do template, informe o nome exatamente como está no template aprovado (esquerda) e o valor a enviar (direita). Use {nome}, {telefone} ou {comunicado} no valor para personalizar com os dados do contato importado na planilha — ou digite um texto fixo se for o mesmo para todos.
Sobre a coluna "comunicado": se a sua planilha (passo 2) tiver uma coluna chamada comunicado, cada contato pode receber um texto diferente — basta deixar o valor da variável como {comunicado} (já configurado acima). Se o comunicado for igual para todo mundo, apague {comunicado} e digite o texto fixo diretamente nesse campo.
⚠ Sobre o rodapé: O footer do template (texto como "AGROGUARD | INTI AGSOLUTIONS") é fixo no template aprovado pela Meta e aparece automaticamente. Não envie ele via API — isso causa erro (#100) Invalid parameter.
Mensagem livre funciona apenas para contatos que enviaram mensagem para você nas últimas 24 horas. Para listas frias, use templates aprovados.
